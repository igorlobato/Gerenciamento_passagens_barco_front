## Funcionalidades:
- Registro: Formulário para criar conta com validação de e-mail.
- Login: Autenticação com e-mail e senha, retornando token JWT.
- Dashboard: Página após login, com opção de logout.
- Segurança: Comunicação com a API usando tokens JWT para autenticação.
- Confirmação de e-mail: Envio de código automático após cadastro para o e-mail com tempo de expiração de 30 minutos.

## Executar com XAMPP ou semelhantes ou com php -S localhost:8001
